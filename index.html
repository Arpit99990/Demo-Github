Git & GitHub Cheat Sheet

A comprehensive guide to using Git and GitHub efficiently.

🔧 Setup

git config --global user.name "Your Name"
git config --global user.email "you@example.com"
git config --list

📂 Create or Clone Repo

git init                # Start a new repo
git clone <url>         # Clone a repo

📄 Basic Commands

  • git status – Check status
  • git add filename – Stage file
  • git add . – Stage all
  • git commit -m "msg" – Commit
  • git log – View commits
  • git diff – Show changes

🔁 Branching

git branch                # List branches
git branch feature        # Create branch
git checkout feature      # Switch to branch
git checkout -b feature   # Create & switch
git merge feature         # Merge to current branch

🌍 Remote Repositories

git remote -v
git remote add origin <url>
git push -u origin main
git pull origin main
git push

🔄 Undo & Reset

  • git restore file – Discard changes
  • git reset HEAD file – Unstage file
  • git commit --amend – Modify last commit
  • git revert <hash> – Revert a commit

🧪 GitHub Workflow

# Common GitHub Workflow
git clone <repo-url>
git checkout -b feature-branch
# make changes
git add .
git commit -m "Added feature"
git push origin feature-branch

# Open Pull Request on GitHub

📘 Markdown Basics

# Heading 1
## Heading 2
**bold**  *italic*  `inline code`
- List item
[Link](https://github.com)

🔗 Useful GitHub Terms

  • Fork: Copy someone else's repo
  • Pull Request: Suggest code changes
  • Issues: Report bugs or request features
  • Actions: Automate CI/CD tasks
  • README.md: Project description file

🔐 SSH vs HTTPS

To use SSH instead of HTTPS:

ssh-keygen -t rsa -b 4096 -C "you@example.com"
# Add public key to GitHub → Settings → SSH Keys
git@github.com:username/repo.git

📦 Git Ignore

Create a .gitignore file to skip files from version control:

node_modules/
.env
.DS_Store
*.log
